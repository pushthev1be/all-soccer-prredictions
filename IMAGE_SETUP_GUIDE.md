## üéØ Quick Setup Guide

### Step 1: Download Images
1. Go to Unsplash.com
2. Search for the recommended terms
3. Download the highest resolution (click "Download free" ‚Üí Select largest size)
4. Rename files as shown above

### Step 2: Add to Project
1. Place all downloaded images in:
   ```
   public/images/backgrounds/
   ```
2. Ensure filenames match:
   - `stadium.jpg`
   - `football-pattern.jpg`
   - `match-action.jpg`
   - `celebration.jpg`

### Step 3: Optimize Images (Optional but Recommended)
- Use TinyPNG.com to compress without quality loss
- Target size: 500KB - 1MB per image
- Keeps page load fast
